# jwmarshall's Claude Code Plugins

A personal [Claude Code](https://claude.com/claude-code) plugin marketplace.

This repository hosts plugins (slash commands, subagents, hooks, and MCP
servers) that I use with Claude Code, packaged so they can be installed via the
marketplace.

## Installation

Add this marketplace in Claude Code:

```
/plugin marketplace add jwmarshall/claude-plugins
```

Then browse and install plugins with:

```
/plugin
```

## Plugins

| Plugin                       | Description                                                                                                                                                   |
| ---------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [scopewright](./scopewright) | Interactively scaffold a SCOPED review subagent tailored to your project — you supply the domain, it prescribes the read-only, injection-resistant structure. |
